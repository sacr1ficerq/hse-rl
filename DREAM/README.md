# DREAM overview

This repo implements basic version of DREAM paper algorithm.

It uses [PokerRL](https://github.com/EricSteinberger/PokerRL) library, introduced by EricSteinberger, most of the code is full version of DREAM, rewritten for simplicity.

Video overview of article by me could be found [here](https://www.youtube.com/watch?v=7IouSDnFl_s).
